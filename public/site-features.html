🍳 RecipeWorld

RecipeWorld — кулинарная платформа, где можно находить рецепты, сохранять идеи, получать рекомендации от ИИ и делиться своими блюдами с сообществом.

Основные функции

  • Поиск и просмотр рецептов по категориям, времени и сложности.
  • Подбор блюд на основе ингредиентов в холодильнике.
  • ИИ-помощник для генерации новых идей и быстрых вариантов блюд.
  • Профиль пользователя с сохраненными рецептами и уведомлениями.
  • Интерактивные карточки рецептов с лайками и авторством.

Для кого сервис

  • Для начинающих: простые и быстрые рецепты на каждый день.
  • Для занятых: фильтры по времени приготовления и ингредиентам.
  • Для энтузиастов: эксперименты с новыми блюдами и подачей.
